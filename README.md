# Wolfy Discord Bot

Wolfy is a private Discord chatbot designed to serve the **LUPOS** group.  
It integrates advanced AI models to provide short, focused, and motivational responses aligned with the LUPOS mindset.

This project is **NOT open source**.

---

## üöÄ Project Overview

Wolfy is a Discord bot built with **Node.js** and **discord.js**, powered by **GitHub-hosted AI models** via Azure AI Inference.

Its purpose is to:
- Act as an intelligent assistant inside a Discord server
- Respond to commands using a custom personality and system prompt
- Reflect the mindset, values, and goals of the **LUPOS brotherhood**
- Deliver concise AI-generated replies suitable for Discord messages

The bot announces itself when it starts or restarts and listens for specific triggers to generate AI responses.
